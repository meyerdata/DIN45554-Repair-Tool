Ein Beispielschema für die Klassifizierung der Kriterien
Kriterien Klassen Werte
Befestigungselemente

A: Leicht entfernbar und wiederverwendbar.

B: Entfernbar, aber nicht wiederverwendbar.

C: Weder Entfernbar noch wiederverwendbar.

A = 5

B = 3

C = 0
Werkzeuge

A: Allgemein verfügbare Werkzeuge (Standard Werkzeuge wie z.B eine Standard Zangen).

B: Spezielle Werkzeuge, die leicht zugänglich sind (z.B. spezielle Zangen).

C: Hersteller-spezifische Werkzeuge (nur vom Hersteller verfügbar).

D: Hochspezialisierte Werkzeuge (nur in spezialisierten Werkstätten verfügbar).

E: Einmalige Werkzeuge (werden nach Gebrauch zerstört oder sind sehr schwer zu beschaffen).

A = 5

B = 4

C = 3

D = 2

E = 0
Verfügbarkeit von Ersatzteilen

A: Öffentlich verfügbar (jeder kann das Ersatzteil erwerben).

B: Für unabhängige Reparaturdienstleister verfügbar (Fachleute, die nicht vom Hersteller autorisiert sind).

C: Für von Herstellern autorisierte Reparaturdienstleister verfügbar.

D: Nur für den Hersteller verfügbar (nur der Hersteller selbst hat Zugang zu den Ersatzteilen).

E: Keine Ersatzteile verfügbar.

A = 5

B = 4

C = 3

D = 2

E = 0
Verfügbarkeitsdauer von Ersatzteilen

A: Langfristige Verfügbarkeit (Ersatzteile sind für einen Zeitraum verfügbar, der die erwartete Lebensdauer des Produkts abdeckt).

B: Mittelfristige Verfügbarkeit (Ersatzteile sind für einen Zeitraum verfügbar, der eine mittlere Lebensdauer des Produkts abdeckt).

C: Kurzfristige Verfügbarkeit (Ersatzteile sind für einen kurzen Zeitraum verfügbar, z.B. zwei Jahre nach Verkauf des Produkts).

D: Keine Informationen zur Dauer der Verfügbarkeit.

A = 5

B = 4

C = 2

D = 1
Verfügbarkeit von Informationen (Umfang)

A: Umfassend (detaillierte und vollständige Reparaturanleitungen, inklusive Schaltpläne, Diagnosesoftware und Teilelisten).

B: Mittel (ausreichende Informationen für grundlegende Reparaturen, aber möglicherweise nicht vollständig).

C: Gering (nur sehr grundlegende Informationen oder teilweise unvollständige Anleitungen).

A = 5

B = 3

C = 1
Diagnose-Support und -Schnittstelle

A: Standardisierte Diagnose-Schnittstellen (z.B. USB).

B: Hersteller-spezifische Diagnose-Schnittstellen.

C: Drahtlose Diagnose-Schnittstellen (z.B. Bluetooth, Wi-Fi).

D: Software-basierte Diagnose-Schnittstellen.

E: Keine Diagnose-Schnittstellen verfügbar.

A = 5

B = 4

C = 3

D = 2

E = 0
Arbeitsumgebung

A: Einfache Umgebung (kann überall durchgeführt werden, keine speziellen Anforderungen).

B: Normale Werkstattumgebung (erfordert grundlegende Werkstattausstattung und -bedingungen).

C: Spezialisierte Werkstattumgebung (erfordert spezialisierte Ausrüstung und Bedingungen).

A = 5

B = 4

C = 3
Fähigkeitsniveau

A: Keine besonderen Fähigkeiten erforderlich (einfache Aufgaben, die von jedem durchgeführt werden können).

B: Grundlegende technische Fähigkeiten erforderlich (z.B. Basiskenntnisse in Elektronik oder Mechanik).

C: Fortgeschrittene technische Fähigkeiten erforderlich (z.B. Kenntnisse in spezifischen Reparaturtechniken).

D: Expertenwissen erforderlich (z.B. tiefgehende Kenntnisse in Elektronik oder spezielle Zertifizierungen).

E: Nur durch den Hersteller durchführbar (spezialisierte Kenntnisse und Werkzeuge erforderlich).

A = 5

B = 4

C = 3

D = 2

E = 1
Zurücksetzen von Passwörtern und Werkeinstellungen

A: Einfache und umfassende Zurücksetzoptionen (einfach zugängliche und vollständig integrierte Funktion).

B: Mittelschwere Zurücksetzoptionen (erfordert einige Schritte oder externe Unterstützung).

C: Schwierige Zurücksetzoptionen (erfordert viele Schritte oder technische Kenntnisse).

D: Keine Zurücksetzoptionen (es gibt keine Möglichkeit, Passwörter oder Werkseinstellungen zurückzusetzen).

A = 5

B = 3

C = 2

D = 1
